## Description

Detective Vigenere has flagged the account **ShadowObserver99** for suspicious online activity. The user appears to be stalking someone and posting cryptic updates.

We’ve recovered **three posts**, and investigators believe a hidden message — possibly a future target — is embedded within them.

Your job: **uncover the hidden message and reveal the truth.**
